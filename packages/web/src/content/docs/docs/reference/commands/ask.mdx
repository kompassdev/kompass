## Purpose

Use `/ask` when you want a direct answer about the current project or codebase.

## Usage

```text
/ask <question>
```

## Typical flow

- load only the relevant repository context
- answer directly instead of drifting into implementation

## Common tools

- `read`
- `grep`
- `task(explore)`
