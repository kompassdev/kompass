## Purpose

Use `/commit-and-push` when you want to move current work through commit creation and branch push in one pass.

## Usage

```text
/commit-and-push [message]
```

## Typical flow

- inspect the current diff
- create a commit with a provided or generated message
- push the branch upstream

## Common tools

- `bash(git status)`
- `bash(git commit)`
- `bash(git push)`
