## Purpose

Use `/commit` to create a git commit from the current worktree.

## Usage

```text
/commit [message]
```

## Typical flow

- inspect staged and unstaged changes
- generate a concise, why-focused commit message when one is not provided
- create the commit and verify the result

## Common tools

- `bash(git status)`
- `bash(git diff)`
- `bash(git commit)`
