## Purpose

Use `/learn` to document durable session discoveries in the most useful `AGENTS.md` scope.

## Usage

```text
/learn [focus]
```

## Typical flow

- review the current session for non-obvious discoveries and repeated friction
- choose the narrowest useful scope such as root or package-level `AGENTS.md`
- read existing guidance before editing to avoid duplication
- add concise learnings and report which files were updated

## Common tools

- `read`
- `apply_patch`
