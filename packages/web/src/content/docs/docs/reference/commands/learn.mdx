## Purpose

Use `/learn` to study patterns in the repository and encode the useful guidance where agents can reuse it.

## Usage

```text
/learn <what-to-learn>
```

## Typical flow

- inspect current conventions in the repo
- summarize reusable rules and practices
- write the guidance into the appropriate `AGENTS.md` surface

## Common tools

- `read`
- `grep`
- `apply_patch`
