## Purpose

Use `/rmslop` to simplify noisy or overly complex code and workflow output.

## Usage

```text
/rmslop
```

## Typical flow

- inspect the branch for unnecessary complexity, duplication, or dead weight
- simplify code and supporting workflow surfaces
- leave the branch smaller and clearer

## Common tools

- `changes_load`
- `read`
- `apply_patch`
- `task(reviewer)`
