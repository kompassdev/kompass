Kompass command templates can include reusable Eta partials from `packages/core/components/`.

Bundled components:

- `change-summary`
- `changes-summary`
- `commit`
- `dev-flow`
- `load-pr`
- `load-ticket`
- `summarize-changes`

## Bundled components

### `change-summary`

Shared partial for loading and summarizing change context.

### `changes-summary`

Shared partial for rendering grouped change summaries.

### `commit`

Shared partial for commit-oriented workflow steps.

### `dev-flow`

Shared partial for implementation-oriented execution flow.

### `load-pr`

Shared partial for loading PR context in review-oriented workflows.

### `load-ticket`

Shared partial for loading ticket context from GitHub, files, or text.

### `summarize-changes`

Shared partial for PR and ticket summary generation.
