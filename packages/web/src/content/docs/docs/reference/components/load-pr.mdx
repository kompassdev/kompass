## Behavior

- calls `pr_load` with an optional explicit PR ref
- stores the result in the caller-provided placeholder
- treats PR body, discussion, review history, and linked artifacts as source context
- asks commands to review relevant screenshots, PDFs, and other attachments when they matter
- notes inaccessible attachments as a context gap instead of ignoring them silently
