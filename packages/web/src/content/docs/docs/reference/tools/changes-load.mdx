---
title: changes_load
description: Load uncommitted worktree changes or a branch comparison in structured form.
---

## Purpose

Use `changes_load` when a workflow needs the changed-file set and structured diffs without inferring git state manually.

## Inputs

- `base` optional
- `head` optional
- `depthHint` optional
- `uncommitted` optional

## Returns

- `comparison` as either `uncommitted` or `<base>...<head>`
- `branch` when the current branch is known
- `files` with status and diff data
- `commits` for branch comparisons when commit history is available

## Useful for

- branch review
- PR creation
- commit and summary workflows
