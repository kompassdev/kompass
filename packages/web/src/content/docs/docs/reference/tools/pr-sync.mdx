---
title: pr_sync
description: Create, update, comment on, or review a GitHub pull request.
---

## Purpose

Use one tool surface for PR creation, metadata updates, comments, replies, and formal review submission.

## Notable inputs

- `title`
- `body`
- `description`
- `base`
- `head`
- `assignees`
- `checklists`
- `draft`
- `refUrl`
- `commitId`
- `review`
- `replies`
- `commentBody`

## Useful for

- `/pr/create`
- `/pr/review`
- `/pr/fix`
