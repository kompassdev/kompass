## Purpose

Use one tool surface for PR creation, updates, comments, replies, and formal review submission.

## Notable inputs

- `title`
- `description`
- `base`
- `head`
- `checklists`
- `draft`
- `review`
- `replies`
- `commentBody`

## Useful for

- `/pr/create`
- `/pr/review`
- `/pr/fix`
